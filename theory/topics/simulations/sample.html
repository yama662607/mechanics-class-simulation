単振り子のシミュレーション

このページでは、ShinyLive を用いたPythonシミュレーションコードを解説します。 以下のコードは、サーバーを立てることなく完全にブラウザ上のWebAssembly（Pyodide）で実行されています。コードの数式やパラメータをその場で書き換えて再実行することも可能です。

インタラクティブ・シミュレーション

スライダーを動かしたり、時間スライダーの横にある「再生ボタン」を押してアニメーションを確認してください。

#| '!! shinylive warning !!': |
#|   shinylive does not work in self-contained HTML documents.
#|   Please set `embed-resources: false` in your metadata.
#| standalone: true
#| components: [editor, viewer]

import numpy as np
import matplotlib.pyplot as plt
from shiny import App, render, ui

# --- UI（ユーザーインターフェース）の定義 ---
app_ui = ui.page_fluid(
    ui.layout_sidebar(
        ui.sidebar(
            ui.input_slider("length", "振り子の長さ L (m)", 0.1, 2.0, 1.0, step=0.1),
            ui.input_slider("initial_angle", "初期角度 θ0 (rad)", -np.pi/2, np.pi/2, np.pi/4, step=0.1),
            ui.input_slider("gravity", "重力加速度 g (m/s²)", 1.0, 20.0, 9.8, step=0.1),
            # アニメーション可能な時間スライダー
            ui.input_slider("time", "時間 t (s)", 0.0, 10.0, 0.0, step=0.1, 
                            animate=ui.AnimationOptions(interval=100, loop=True))
        ),
        ui.output_plot("pendulum_plot"),
    )
)

# --- サーバー（計算ロジック）の定義 ---
def server(input, output, session):
    @render.plot(alt="単振り子のシミュレーション")
    def pendulum_plot():
        # スライダーから入力値を取得
        L = input.length()
        g = input.gravity()
        theta0 = input.initial_angle()
        t = input.time()
        
        # 単振動の角振動数と角度の計算
        omega = np.sqrt(g / L)
        theta = theta0 * np.cos(omega * t)
        
        # (x, y) 座標へ変換（下向きを負とする）
        x = L * np.sin(theta)
        y = -L * np.cos(theta)
        
        # プロットの初期化
        fig, ax = plt.subplots(figsize=(6, 6))
        ax.set_xlim(-2.5, 2.5)
        ax.set_ylim(-2.5, 0.5)
        ax.set_aspect('equal')
        ax.grid(True)
        ax.set_title(f"時間 t = {t:.1f} s における単振り子の位置")
        
        # 支点、紐、おもりを描画
        ax.plot(0, 0, 'ko', markersize=8)        # 支点
        ax.plot([0, x], [0, y], 'k-', lw=2)      # 紐
        ax.plot(x, y, 'ro', markersize=20)       # おもり
        
        return fig

# アプリの起動
app = App(app_ui, server)

理論的背景

長さ l の軽い糸に質量 m のおもりをつるした単振り子の運動方程式は、以下のようになります。

\frac{d^2 \theta}{dt^2} = -\frac{g}{l} \sin \theta

微小振動（\theta \ll 1）の近似を用いると、\sin \theta \approx \theta となり、方程式は単振動の形になります。

\frac{d^2 \theta}{dt^2} = -\frac{g}{l} \theta

この方程式の一般解は以下の通りです。

\theta(t) = \theta_0 \cos\left(\sqrt{\frac{g}{l}} t + \phi \right)
