update_time = {

    viewof time.dispatchEvent(new CustomEvent("input"));
    await Promises.delay(100);
  }
}

理論的背景

長さ l の軽い糸に質量 m のおもりをつるした単振り子の運動方程式は、以下のようになります。

\frac{d^2 \theta}{dt^2} = -\frac{g}{l} \sin \theta

微小振動（\theta \ll 1）の近似を用いると、\sin \theta \approx \theta となり、方程式は単振動の形になります。

\frac{d^2 \theta}{dt^2} = -\frac{g}{l} \theta

この方程式の一般解は以下の通りです。

\theta(t) = \theta_0 \cos\left(\sqrt{\frac{g}{l}} t + \phi \right)

```{pyodide}
    with open("/tmp/NotoSansJP.ttf", "wb") as f:
        f.write(await resp.bytes())
    font_manager.fontManager.addfont("/tmp/NotoSansJP.ttf")
    matplotlib.rcParams["font.family"] = "Noto Sans JP"
    matplotlib.rcParams["axes.unicode_minus"] = False

# --- 描画ロジック ---
import numpy as np
import matplotlib.pyplot as plt

# 単振動の角振動数と角度の計算 (小振動近似)
omega = np.sqrt(gravity / length)             # 角振動数
theta = initial_angle * np.cos(omega * time)  # 角度 θ(t)

# 直交座標変換
x = length * np.sin(theta)
y = -length * np.cos(theta)  # 支点が原点、下向き負

fig, ax = plt.subplots(figsize=(6, 6))
ax.set_xlim(-2.5, 2.5)
ax.set_ylim(-2.5, 0.5)
ax.set_aspect("equal")
ax.grid(True)
ax.set_title(f"時間 t = {time:.1f} s における単振り子の位置")

ax.plot(0, 0, "ko", markersize=8)        # 支点
ax.plot([0, x], [0, y], "k-", lw=2)      # 糸
ax.plot(x, y, "ro", markersize=20)       # おもり

plt.tight_layout()
plt.show()
```
